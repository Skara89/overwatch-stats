Overwatch Tier List

Persönliches Tool, das Heldenstatistiken live von der OverFast API lädt (inoffizielle, auf Blizzards eigenen Daten basierende API). Einstufung kombiniert Win-Rate und Pick-Rate; Ergebnisse werden im Browser zwischengespeichert und stündlich neu geladen. Blizzard liefert nur den kumulierten Wert seit Patch-Start (keine echten Zeitfenster) — der optionale Trend-Filter zeigt daher die Veränderung gegenüber einem älteren, täglich gespeicherten Snapshot, nicht eine exakte Win-Rate "nur der letzten N Tage". Nur bei ungefilterten Rängen/Rollen/Maps verfügbar.
