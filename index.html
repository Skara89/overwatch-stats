Overwatch Hero Stats

ℹ️ Über die Daten
Lädt Heldenstatistiken live von der OverFast API (inoffiziell, basiert auf Blizzards eigenen Daten). Die Einstufung kombiniert Win-Rate und Pick-Rate; der Balken zeigt die Abweichung von 50 % Win-Rate. Blizzard liefert nur kumulierte Werte seit Patch-Start — der optionale Trend-Filter zeigt daher die Veränderung gegenüber einem älteren, täglich gespeicherten Snapshot, keine exakte Win-Rate „nur der letzten N Tage", und ist nur ohne Rang-/Rollen-/Map-Filter verfügbar.
